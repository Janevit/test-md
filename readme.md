5. เปิดแอป Company Portal
6. กดปุ่ม "Sign in" เพื่อเข้าสู่ระบบ
7. ป้อนรายละเอียดบัญชีผู้ใช้ของคุณ
8. กด "ถัดไป" เพื่อดำเนินการต่อ
